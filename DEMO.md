# Signal Desk Demo Script

Run the demo:

```bash
python3 scripts/generate_demo_corpus.py --reset
python3 server.py
```

Open `http://localhost:3000`.

## Recommended Flow

1. `What is the Growth team working on?`
   - Proves team-level synthesis across multiple projects.

2. `What is Priya working on?`
   - Proves person lookup across Slack, Jira, and Drive records.

3. `What changed since yesterday?`
   - Proves recent-change and stale-plan handling.

4. `What blockers are slowing down the billing migration?`
   - Proves blocker detection and cited evidence.

5. `What decisions were made about the enterprise analytics launch?`
   - Proves decision lookup from Slack and docs.

6. `Which Drive docs look stale compared with Slack or Jira?`
   - Proves stale documentation detection.

7. `Who looks overloaded?`
   - Proves cross-project workload signal, especially Mina Torres.

8. `Which projects have cross-team dependencies?`
   - Proves dependency detection between Growth and Platform.

9. `What should I ask Taylor in our 1:1?`
   - Proves manager workflow use case.

10. `What should I bring up in staff?`
    - Proves executive/staff-meeting summary behavior.

## What To Point Out

- The top-right corpus counter shows the fake indexed corpus size.
- Answers are generated by OpenAI when `mode: openai-responses` appears.
- Citation chips map to inline source IDs like `[S1]`.
- Source cards include source type, team/project, and evidence snippets.
